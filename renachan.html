tracking

renachan has tracking functionality, using the command !rena track will save the contents to the database and will inform you if the price changes!

currently:

  • !rena track [id/class] [id/class name here] [currency symbol]
  • track works in two ways, by tracking the class or id of an html content
  • renachan uses a custom crawl class that utilizies beautiful soup to extract important html from html pages
  • it's intentions are to find the number value next to the currency symbol so the id or class name you need is the one associated with the currency symbol
  • if there are more than one possible values extracted then renachan will use a bfs to find the div associated with the item to get the correct price
harass

renachan has a harass feature, feel free to message her with a message, user, as well as password to ensure you get reached out to!

currently:

  • !rena harass [user] [password] [message]
  • renachan will continously ping a user you share a server with a message of your choice
  • the only way to get the messages to stop (without blocking of course) is to give renachan the password you sent!
  • this ensures you will always be reached out to
